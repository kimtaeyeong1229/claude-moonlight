## 1. OS별 실행 방식 분기 (설치 시)
- 설치/실행 시 **운영체제를 파악**해서:
  - **macOS** → 지금처럼 **네이티브 앱**(`/Applications/claude-moonlight.app`, WKWebView).
  - **Windows / Linux** → 앱 대신 **웹으로 실행**(컨테이너 기동 후 브라우저로 `http://127.0.0.1:8090` 열기).
- 이유: PTY/fork/launchd/swiftc 기반 네이티브 앱은 macOS 전용. 다른 OS는 컨테이너 + 브라우저가 현실적.
- 작업 메모:
  - `make-app.sh`(또는 통합 `install` 스크립트)에서 `uname`/OS 감지 → 분기.
  - Windows/Linux 용 런처: 컨테이너 `up` + 기본 브라우저 open(`xdg-open`/`start`).
  - 단, 번역 워커는 여전히 **호스트 claude 인증** 필요 → OS별 claude 위치/실행 방식 점검.
